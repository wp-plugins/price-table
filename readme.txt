== Description ==


awesome pricing table by easy admin panel to drag and drop row column and delete any row or columns and update.
use via shortcode anywhere with some aswesome pricing table themes.


### Price Table by http://paratheme.com

<br />
* [See the Live demo!&raquo;](http://paratheme.com/demo/price-table/)


<strong>Plugin Features</strong><br />

* Use Via Short-codes.
* Unlimited row columns.
* Drag & Drop row, columns.
* Add new row, column.
* Any Google Fonts.
* Coulmn Custom Width.
* 19 Ready Coulmn Ribbons.
* Coulmn Video or Images(Youtube & vimeo).
* Background image for table area.
* Different Themes.
* 12 Ready Background Color Pack.
* 7 Ready Text Color Pack.
* Custom Price Duration Text.




== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Price Table</strong>" activate it.<br />

After activate plugin you will see "Price Table" menu at left side on WordPress dashboard click "New Price Table" and use the options field "Price Table Options"<br />

<br />
<strong>How to use on page or post</strong><br />
When Price Table options setup done please publish Price Table<br />

and then copy shortcode from top of <strong>Price Table Options</strong> `[price_table id="1234" ]`<br />

then paste this shortcode anywhere in your page to display Price Table<br />
