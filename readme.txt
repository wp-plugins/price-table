== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Price Table</strong>" activate it.<br />

After activate plugin you will see "Price Table" menu at left side on WordPress dashboard click "New Price Table" and use the options field "Price Table Options"<br />

<br />
<strong>How to use on page or post</strong><br />
When Price Table options setup done please publish Price Table<br />

and then copy shortcode from top of <strong>Price Table Options</strong> `[price_table id="1234" ]`<br />

then paste this shortcode anywhere in your page to display Price Table<br />
